# GoToServicePortalFormField

### Usage in a template

You can create a GoToServicePortalFormField using the following function `GoToServicePortalFormField`.

<Source
  language="typescript"
  code={dedent(`
    GoToServicePortalFormField({
      id: 'field.id',
      title: '',
    })`)}
/>

The previous configuration object will result in the following component:

<Canvas>
  <Story name="Default">
    <GoToServicePortalFormField
      application={createMockApplication()}
      field={{
        id: 'field.id',
        title: '',
      }}
    />
  </Story>
</Canvas>

You can also use this field into a custom component by using `<GoToServicePortalFormField field={...} />` with the configuration object defined above.
