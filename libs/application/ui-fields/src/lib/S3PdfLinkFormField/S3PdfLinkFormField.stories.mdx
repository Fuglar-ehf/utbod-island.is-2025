# S3PdfLinkFormField

### Usage in a template

You can create a S3PdfLinkFormField using the following function `S3PdfLinkFormField`.

<Source
  language="typescript"
  code={dedent(`
    S3PdfLinkFormField({
      id: 'field.id',
      title: 'Download PDF',
      s3Key: 'lorem_ipsum_pdf_key',
    })`)}
/>

The previous configuration object will result in the following component:

<Canvas>
  <Story name="Default">
    <S3PdfLinkFormField
      application={createMockApplication()}
      field={{
        id: 'field.id',
        title: 'Download PDF',
        s3Key: 'lorem_ipsum_pdf_key',
      }}
    />
  </Story>
</Canvas>

You can also use this field into a custom component by using `<S3PdfLinkFormField field={...} />` with the configuration object defined above.
