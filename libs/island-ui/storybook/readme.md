# island-ui-storybook

`island-ui-storybook` is a UI component library generated with [Nx](https://nx.dev), a smart and extensible build framework. This library facilitates efficient building and testing of UI components.

## Running Unit Tests

To run the unit tests for this project using [Jest](https://jestjs.io), enter the following command in your terminal:

```bash
nx test island-ui-storybook
```
