### Custom links inside text

Depending on which React router we're using we often need to control how links are managed.

```tsx
import { Link } from 'next/Link'

;<Text
  linkRenderer={(href, children) => (
    <Link
      href={{
        pathname: href,
        query: { slug: 'something' },
      }}
    >
      <a>{children}</a>
    </Link>
  )}
>
  This a <a href="/abc">link</a> that will be rendered as a Next Link
</Text>
```

#### Example:

<Canvas>
  <Story name="Custom link renderer">

          {children}
        </Text>
      )}
