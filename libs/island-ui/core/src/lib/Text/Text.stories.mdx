### Custom links inside text

Depending on which React router we're using we often need to take control of how links are rendered.
Wrap your app in `<LinkContext.Provider />` and provide a custom renderer function.

#### Example:

<Canvas>
  <Story name="Custom link renderer">
    <LinkContext.Provider
      value={{
        linkRenderer: (href, children) => (
          <a
            style={{
              color: 'deeppink',
              fontWeight: 500,
              textShadow: '1px 1px 0 rebeccapurple',
            }}
            href={href}
            rel="noopener noreferrer"
            target="_blank"
          >
            {children + 'âœ¨'}
          </a>
        ),
      }}
    >
      <Text>
        Here's a <a href="/test">custom link</a> and here's{' '}
        <a href="/test2">another</a>.
      </Text>
    </LinkContext.Provider>
  </Story>
</Canvas>
