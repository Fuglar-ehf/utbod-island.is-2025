# Our MDX Template

### With `MDX` we can define a story for our `ExampleComponent` right in the middle of our Markdown documentation.

### We can drop it in a `Canvas` to get a code snippet:

<Canvas>
  <Story name="Example story">
    <ExampleComponent />
  </Story>
</Canvas>

### Or render it directly:

<Box paddingY={1}>
  <ExampleComponent exclamationPoint />
</Box>

### And since this is markdown we can print code:

```tsx
<ExampleComponent exclamationPoint={true} />
```
