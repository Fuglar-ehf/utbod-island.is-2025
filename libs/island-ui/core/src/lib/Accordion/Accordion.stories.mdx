# Accordion

[View in Figma](https://www.figma.com/file/pDczqgdlWxgn3YugWZfe1v/UI-Library-%E2%80%93-%F0%9F%96%A5%EF%B8%8F-Desktop?node-id=208%3A446)

The **Accordion** component has three different type of items that can be rendered.

- `<AccordionItem />` rendered inside the **Accordion** component
- `<AccordionCard />` and `<SidebarAccordion />` rendered as standalone

<br />

### Basic usage

We wrap all the `<AccordionItem />` inside a `<Accordion />` component.
