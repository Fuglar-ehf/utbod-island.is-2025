# Island UI Theme

The Island UI Theme library was created using [Nx](https://nx.dev). It provides a customizable UI theme to maintain design consistency across various applications.

## Running Unit Tests

To execute unit tests for the Island UI Theme library, follow these steps:

1. Ensure all required dependencies are installed.
2. Use the following command to run the tests:

   ```bash
   nx test island-ui-theme
   ```

This command runs the tests using [Jest](https://jestjs.io), a popular JavaScript testing framework.
