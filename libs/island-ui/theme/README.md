# Island UI Theme

The Island UI Theme library was generated with [Nx](https://nx.dev). It provides a customizable UI theme to maintain design consistency across various applications.

## Running Unit Tests

To execute unit tests for the Island UI Theme library, use the following command:

```bash
nx test island-ui-theme
```

This command runs the tests with [Jest](https://jestjs.io), a popular JavaScript testing framework. Ensure all required dependencies are installed before running the tests.
