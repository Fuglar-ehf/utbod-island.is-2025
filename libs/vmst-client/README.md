# VMST Client

## About

This library implements a client to use VMST APIs

### Import into other NestJS modules

Add the service to your Module providers:

```typescript
import { VMSTClientModule } from '@island.is/vmst-client'

@Module({
  imports: [VMSTClientModule],
})
```
