The `judicial-system-auth` library facilitates authentication within the judicial system application.

This library was generated using the [Nx](https://nx.dev) build system, enabling efficient and scalable development processes.

## Running Unit Tests

To execute the unit tests for this library, run the following command:

```bash

This command leverages [Jest](https://jestjs.io) to execute the test suite, ensuring correctness and reliability of the library's functionality.
