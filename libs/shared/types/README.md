# Introduction

The purpose of this library is to store shared types used throughout various libraries and applications. This includes common ElasticSearch type definitions.

## Usage

Integrate this library into your project to maintain consistency in type definitions across multiple modules. This ensures that different parts of your system adhere to a unified data schema, reducing errors and enhancing maintainability.

## ElasticSearch Types

This library includes type definitions that are commonly used when interacting with ElasticSearch. By using these predefined types, developers can ensure that their ElasticSearch queries and operations are type-safe and align with expected data formats.

## Contribution

Feel free to contribute by adding or modifying type definitions as necessary. Please ensure that any contributions maintain compatibility with existing types and provide adequate documentation for new additions.
